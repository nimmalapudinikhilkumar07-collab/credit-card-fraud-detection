# Credit Card Fraud Detection

This project builds a machine learning model to detect fraudulent credit card transactions.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Models Used
- Logistic Regression
- Random Forest

## Evaluation Metrics
- Precision
- Recall
- ROC-AUC
- Confusion Matrix

## Goal
The goal of this project is to identify fraudulent transactions while minimizing false negatives.

Credit Card Fraud Detection Dataset
